###  원리
1. 마크다운 문법 처리 후 HTML 문법 처리.
2. ToC는 마크다운 문법 처리 단계에 해당.
3. Markdown 구문으로 작성된 heading은 ToC 플러그인이 처리.
4. HTML 구문으로 작성된 heading은 ToC 플러그인이 처리하지 못 함.
5. HTML의 `details`로 감춰져 있는 지 여부는 영향 없음.
