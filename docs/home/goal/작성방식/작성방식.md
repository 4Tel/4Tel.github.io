# 작성 방식
* 블로그 작성 시 각종 기준.
* 기준을 따를 것을 권장하나, 따르지 않을 수 있다.
## 언어 설정
* 한국어만 지원
    * 부족한 영어 실력으로 인한 오해 방지
    * 영어 번역을 위한 시간 할애 불가.
* 가능하면 영어를 한국어로 번역해 사용.
* 번역이 부자연스러운 경우 영단어 그대로 사용.
## 분류 위치
* 대분류: 상단바에 위치시킨다.
* 중분류: 게시글의 카드 형식으로 위치시킨다.
* 소분류: 좌측 사이드바에 위치시킨다.
## [구성 방식](구성방식.md)
* 요약 정보 배치
* 개요 작성
* 부록 작성
## [설명 방식](설명방식.md)
* 요약 정보는 하향식 설명 사용.
* 상향식 설명으로 게시글 구성
## 내용 작성
* 학습자는 충분한 지능을 보유함.
* 검색하거나 제공된 자료로 학습할 수 있음.
* 설명이나 문장을 축약하여도 내용 완성이 가능함.
