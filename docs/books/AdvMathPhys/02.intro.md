# 개요
* sin, cos, tan와 같은 삼각함수(trigonometric functions)를 이용하여 주기적인 현상을 모델링할 수 있다.
* 푸리에 급수(Fourier series)를 이용하여 주기적인 함수를 삼각함수의 합으로 표현할 수 있다.
* 주기적인 현상의 예
  * 파동(wave)
  * 회전 기계(Rotary machines)
  * 조화 운동(harmonic motion)
  * 반복 운동(repetitive motion)

### 삼각함수 꼴
