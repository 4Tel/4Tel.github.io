# 원격
## 추가
```bash
git remote add origin https:github.com/(계정)/(저장소)
```
## 확인
```bash
git remote get-url origin
```
또는
```bash
git remote show origin
```
## 분기 확인
```bash
git branch -r
```
## 분기 사용
```bash
git checkout -b (local name) origin/(remote name)
```
## 삭제
```bash
git remote remove origin
```
