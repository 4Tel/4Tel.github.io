# Git
### 초기 생성
```bash
git init
```
## 분기(branch)
### 목록
```bash
git branch
```
또는
```bash
git branch -v
```
### 생성
```bash
git branch (분기명)
```
### 생성 및 이동
```bash
git checkout -b [분기명]
```
### 이동
```bash
git checkout (분기명)
```
### 이름 변경
```bash
git branch -m (old) (new)
```
### 삭제
```bash
git branch -D (분기명)
```
