# 로컬+원격
## 원격 git 가져오기
```bash
git fetch origin
```
## 원격 git 반영하기
```bash
git pull origin main
```
또는
```bash
git pull origin main --rebase
```
또는
```bash
git pull origin main --rebase --autostash
```
## 로컬 git 유지
```bash
git stash
```
## 로컬 git 가져오기
```bash
git stash pop
```
## 충돌 해결
```bash
git add <file>
```
또는 
```bash
git merge --continue
```
