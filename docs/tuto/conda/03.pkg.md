# 패키지
## 보유 목록
```bash
conda list
```
## 설치 가능 버전
```bash
conda search -f (pkgName)
```
## 설치
### 채널 미지정 설치
```bash
conda install (pkgName)
```
### 채널 지정 설치
```bash
conda install -c conda-forge (pkgName)
conda install conda-forge::(pkgName)
```
## 삭제
```bash
conda uninstall (pkgName)
```
## 업데이트
```bash
conda update --all
```
