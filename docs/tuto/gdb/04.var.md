# 변수
## 변수 목록
```sh title="전체 변수 출력"
info variables
```
또는
```sh title="키워드 검색"
info variables (검색 키워드)
```
## 변수 출력
```sh
p (변수명)
```
```sh title="hex type"
p/x (변수명)
```
## 변수 추적
* 변수 읽기/쓰기 시 break
```sh
awatch (변수명)
```
## type cast
```sh title="mp 컴파일"
* (real_8 *) (modulename)_mp_(varname)_@2
```
